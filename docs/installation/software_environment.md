# 软件环境

软件环境章节包括开发需要用到的基础软件、版本约束及IDE推荐。

## Python 与 Pip

由于CARLA发行版指定了Python版本，使用其他版本的Python会导致引用问题，本文档指定使用`Python 2.7`作为Python的版本。

> [!WARNING]
> `Python`与`Pip`的版本需要匹配，使用`python -V`以及`pip -V`查询
